# 🚀 AI Market Analysis API

An intelligent FastAPI-based backend application that fetches real-time
news data and generates AI-powered market insights for different
industry sectors.

## 📌 Project Overview

The AI Market Analysis API analyzes market trends using live news and AI
models.

## ✨ Features

-   Fetch real-time news
-   AI-powered analysis
-   Structured reports
-   FastAPI backend

## 🛠️ Tech Stack

-   Python
-   FastAPI
-   News API
-   Gemini API

## ▶️ Run

python -m uvicorn main:app --reload
